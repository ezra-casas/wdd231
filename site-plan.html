Time Tracker Website Plan

Site Name

Name: TrackCraft

This name conveys the idea of mastering the craft of time management. It appeals to freelancers, consultants, and independent workers who take pride in efficiently tracking their time and projects.

Optional domain availability: trackcraft.app

Site Purpose

The site helps self-employed individuals efficiently track and manage their project time. It provides a simple yet powerful interface to start, stop, and categorize tracked sessions, export logs, and visualize time spent on different tasks.

Scenarios

  • How can I track how many hours I worked for a specific client this week?
  • Is there a way to export my time logs into a CSV file for invoicing?

Color Schema

  • Primary Blue (#2a5d9f) – used for headers, navigation background, buttons
  • Light Gray (#f4f6f8) – used for page background
  • White (#ffffff) – used for cards, content blocks
  • Accent Green (#4CAF50) – used for success messages, start timer button

Typography

  • Montserrat – used for headers and navigation (modern and bold)
  • Roboto – used for body text (clean and highly readable)
